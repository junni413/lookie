## 🧩 Commit Message (Conventional Commit)
<!-- 아래 형식 중 하나로 작성해주세요 -->
<!-- 예: feat: etf 데이터 엑셀 로드 기능 추가 -->

> 타입 목록

| type | 설명 |
| --- | --- |
| feat | 새로운 기능 추가 |
| fix | 버그 수정 또는 typo |
| refactor | 리팩토링 |
| design | CSS 등 사용자 UI 디자인 변경 |
| comment | 필요한 주석 추가 및 변경 |
| style | 코드 포맷팅, 세미콜론 누락 등 코드 로직 변경이 없는 경우 |
| test | 테스트 코드 추가, 수정, 삭제 (비즈니스 로직 변경 없음) |
| chore | 위에 해당하지 않는 기타 변경사항 (빌드 스크립트 수정, assets image, 패키지 매니저 등) |
| rename | 파일 또는 폴더명 수정 및 이동 |
| remove | 파일 삭제만 수행한 경우 |
| docs | 문서 작성/수정 |
| build | 빌드 파일 수정 |
| ci | CI 설정 파일 수 |
| set | 프로젝트 환경 설정 |
